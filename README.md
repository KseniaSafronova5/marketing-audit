#  Analytics Ecosystem: От данных к автоматизации

Комплексное решение для малого и среднего бизнеса по автоматизации маркетинга и аналитики без штатного специалиста.

## Обзор сервисов
Проект развернут на собственном облачном сервере с использованием Docker и Nginx Proxy Manager.

* **[Аналитика без аналитика](https://analytika.ksautomationai.com)** — Инструмент для глубокого анализа данных и визуализации ключевых метрик.
* **[Маркетинг Аудит](https://marketing.ksautomationai.com)** — Автоматизированная проверка эффективности маркетинговых кампаний.

##  Технологический стек
* **Язык**: Python (Streamlit для интерфейса, Pandas для обработки данных).
* **Инфраструктура**: Docker & Docker Compose (изоляция сервисов).
* **DevOps**: Nginx Proxy Manager (SSL-шифрование, управление поддоменами).
* **Автоматизация**: n8n (интеграция сценариев и вебхуков).

## Архитектура проекта
Каждый сервис работает в отдельном Docker-контейнере, что обеспечивает стабильность и независимость обновлений:
1. **Analytika Service**: Контейнер на порту 8501.
2. **Marketing Service**: Контейнер на порту 8502 с подключенными Volumes для динамического обновления данных.

## Ключевые возможности
* **Автономность**: Система работает 24/7 на выделенном сервере.
* **Безопасность**: Все соединения защищены SSL-сертификатами Let's Encrypt.
* **Масштабируемость**: Возможность быстрого развертывания новых аналитических модулей через поддомены.

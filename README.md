# Маркетинг-аудит: Автоматизированный контроль рекламных кампаний

**Ссылка на проект:** [https://marketing.ksautomationai.com/](https://marketing.ksautomationai.com/)

## Описание проекта
Интерактивное веб-приложение на базе **Streamlit**, разработанное для оперативного аудита маркетинговых показателей из облачных источников. 

Проект реализует полный цикл мониторинга: от импорта данных из Google Таблиц до визуализации воронки продаж и расчета эффективности каналов в реальном времени. Цель проекта — предоставить владельцам бизнеса прозрачный инструмент для контроля маркетингового бюджета и выявления "точек роста" без необходимости ручного сведения таблиц.

## Ключевые возможности
* **Сквозная аналитика:** Автоматический расчет метрик CAC, ROAS и ROMI по всем каналам трафика.
* **Визуализация воронки:** Наглядное представление конверсии на каждом этапе — от клика до покупки.
* **Умный анализ:** Мгновенная сегментация данных по датам, рекламным кампаниям и выявление наиболее эффективных каналов привлечения.
* **Масштабируемость:** Гибкая архитектура позволяет подключать новые источники данных и настраивать кастомные дашборды.

## Безопасность и конфиденциальность
* **Защищенный канал:** Доступ к приложению осуществляется по протоколу HTTPS через Nginx Proxy Manager.
* **Принцип "In-Memory":** Обработка данных происходит в оперативной памяти сервера без сохранения промежуточных файлов на диск.
* **Изоляция:** Приложение развернуто в контейнере Docker, что гарантирует стабильность и безопасность окружения.

## Стек технологий
* **Python 3.x / Streamlit:** Фронтенд и интерактивный интерфейс.
* **Pandas:** Обработка, очистка и трансформация массивов данных.
* **Docker:** Контейнеризация и управление инфраструктурой.
* **Nginx Proxy Manager:** Управление обратным прокси и SSL-сертификатами.
* **Google Sheets API:** Интеграция с внешними источниками данных.

## Как запустить
1. Клонируйте репозиторий.
2. Установите зависимости: `pip install -r requirements.txt`
3. Запустите приложение через Docker: `docker build -t marketing-app .`
4. Запустите контейнер: `docker run -p 8502:8501 marketing-app`

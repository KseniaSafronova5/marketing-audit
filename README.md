#  Маркетинг-аудит: Автоматизированный контроль рекламных кампаний

**Live demo:** [marketing.ksautomationai.com](https://marketing.ksautomationai.com/)

Интерактивное веб-приложение для оперативного аудита маркетинговых показателей из облачных источников. От импорта данных из Google Таблиц до визуализации воронки продаж и расчёта эффективности каналов в реальном времени.

> Цель: дать владельцам бизнеса прозрачный инструмент контроля маркетингового бюджета без ручного сведения таблиц.

---

##  Ключевые возможности

- **Сквозная аналитика** — автоматический расчёт CAC, ROAS и ROMI по всем каналам трафика
- **Визуализация воронки** — конверсия на каждом этапе от клика до покупки
- **Умный анализ** — сегментация по датам, кампаниям, выявление наиболее эффективных каналов
- **Масштабируемость** — гибкая архитектура для подключения новых источников данных

##  Стек технологий

| Слой | Технология |
|---|---|
| Приложение | Python 3.x · Streamlit |

| Инфраструктура | Docker · Nginx Proxy Manager · HTTPS |

---

##  Безопасность

- **HTTPS** — доступ через Nginx Proxy Manager с SSL
- **In-Memory** — данные обрабатываются в RAM, не сохраняются на диск
- **Docker-изоляция** — контейнеризированное окружение

##  Как запустить локально

```bash
# 1. Клонировать репозиторий
git clone https://github.com/KseniaSafronova5/marketing-audit.git
cd marketing-audit

# 2. Установить зависимости
pip install -r requirements.txt

# 3. Запустить приложение
streamlit run app.py
```

Или через Docker:

```bash
docker build -t marketing-audit .
docker run -d -p 8502:8501 --name marketing-app marketing-audit
```

Приложение будет доступно на `http://localhost:8502`
